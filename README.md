Tactical RPG with resource gathering, crafting, and combat. Playable in the browser.

## Quick Start (Browser)

```bash
pnpm install
pnpm dev
# Open http://localhost:3333
```

## Scripts

| Command | Description |
|---------|-------------|
| `pnpm dev` | Start local server |
| `pnpm lint` | Run Biome linter |
| `pnpm lint:fix` | Auto-fix lint issues |
